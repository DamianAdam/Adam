# 科学上网搭建记录
## 写在前面
这里只是个人使用VPS自建科学上网工具时的一些学习记录，需要的朋友可以进行相关学习。但切记出门在外，该守的规矩还是得守，以免给自己招来一些不必要的麻烦。
